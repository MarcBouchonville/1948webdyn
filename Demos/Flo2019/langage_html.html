Langage html

Accueil   Langage de programmation  Exercices


Le langage html(HyperText Markup Language) est un langage de balisage introduit en 1993, il définit le contenu d'une page. Ainsi, tous les sites que vous visitez sont écris en html (ou en d'autres langages spécifiques). Ce langage est facile à utiliser lorsque l'on connait les balises. Quelques une de celles-ci sont décrites et expliquées ci-dessous. De plus, si vous désirez habiller votre page, il existe un langage lié à l'html. Celui-ci s'appelle le langage CSS (Cascading Style Sheet) apparu en 1997.


Balises de base


Balises par paire

  • ‹html› : ouverture d'une page html
  • ‹head› : zone d'en-tête du fichier HTML
  • ‹title› : titre générique du document Web
  • ‹body› : corps du document
  • ‹header› : reprend l'en-tête de la page
  • ‹section› : définit la zone dans laquelle sera traité un même sujet ou une même fonctionnalité
  • ‹footer› : définit le pied de page
  • ‹h1›, h2, h3, h4, h5 et h6 : permettent d'identifier les titres et le numéro leur donne une importance différente
  • ‹p› : paragraphe
  • ‹nav› : balise de section qui contient la navigation
  • ‹a› : Un lien hypertexte peut aussi bien être placé sur du texte que sur une image ou les deux à la fois.
  • href = "" : attribut qui contient un url. Celui-ci peut être basé soit sur l'adressage relatif(C'est le chemin à parcourir entre le fichier appelant (là où on introduit le lien) et le fichier appelé (celui qui s'ouvre)) soit sur l'adressage absolu(url complète d'un fichier)
  • ‹link› : Permet de lier une feuille de styles à un document HTML. Doit être inséré dans la partie "head" du document HTML.
  • rel = "" : attribut qui précise que c'est une feuille de style
  • type = "" : attribut qui définit le type de feuille de style
  • href = "" : attribut qui contient le lien vers la feuille de style

Balises orphelines

  • ‹br› : retour à la ligne
  • ‹hr› : ligne horizontale

Balises de styles


Balises par paire

  • ‹b› : met en gras
  • ‹i› : met en italique
  • ‹u› : souligne
  • ‹style› : insérer un code css dans le corps html

Balises de structure

Listes

  • ‹ul› : liste à puces
  • ‹ol› : liste ordonnée

Tableaux

  • ‹table border=""› : ouverture d'un tableau, la balise border définit si le tableau doit être encadré(border=1) ou non(border=0)
  • ‹caption› : balise qui permet de légender le tableau. Doit être placée tout de suite après l'ouverture du tableau
  • ‹tr› : définit les lignes du tableau
  • ‹td› : définit la ligne du tableau
  • ‹thead› : définit l'en-tête du tableau qui comprendra les intitulés de colonnes
  • ‹th› : définit l'en-tête, fonctionne comme la balise td
  • ‹tbody› : balise de corps de tableau qui reprendra toutes les lignes qui comprennent les données

Formulaires

  • ‹form› : délimite la zone du formulaire
  • method = "" : attribut qui signale au serveur de quelle manière sont passées les informations
  • action = "" : paramètre qui signale au serveur l'adresse du script qui va traiter les informations encodées. Le script est un CGI ou un fichier php,...
  • ‹input› : permet d'insérer un champ dans le formulaire
  • type = "" : attribut de l'input. Ces valeurs peuvent être : text, radio, checkbox, password, submit, email, ...
  • value = "" : attribut qui dé la valeur du champ
  • ‹textarea› : permet de disposer d'une zone de texte plus ou moins grande où l’utilisateur peut écrire un texte plutôt qu’une phrase.
  • ‹select› : permet d'insérer une liste déroulante
  • ‹option› : permet d'ajouter une option dans la liste déroulante
  • name = "" : attribut qui s'applique à tous les champs et qui précise le nom d'un champ. Ce nom est utilisé par les scripts lors de la récupération des données.

Images

  • ‹img› : insère une image
  • src = "" : attribut qui précise l'url de l'image source
  • alt = "" : attribut qui donne un nom équivalent à l'image lorsque celle-ci n'est pas prise en compte par le serveur
  • width = "" : permet d'indiquer la largeur de l'image, mesurée en pixel (px) ou en pourcentage (%)
  • height = "" : permet d'indiquer la hauteur de l'image, mesurée en pixel (px) ou en pourcentage (%)
  • ‹figure› : rassemble les images et les légendes qui y sont associées
  • ‹figcaption› : permet de placer une légende

Contenus multimédias

  • ‹audio› : permet d'insérer un fichier audio
  • ‹video› : permet d'insérer une vidéo
  • controls : attribut des balises "audio" et "video" assurant la présence des contrôles de lecture
  • ‹iframe› : permet d'intégrer du contenu externe dans vos pages
  • src = "" : attribut qui précise l'url du fichier audio ou de la vidéo
  • height = "" : attribut vous permet de définir une valeur en pixels définissant la hauteur de l'iframe
  • width = "" : attribut vous permet de définir une valeur en pixels définissant la largeur de l'iframe

Pour les plus curieux, vous pouvez naviguer vers le site suivant https://www.w3schools.com/. Celui-ci vous donnera plus d'informations.
